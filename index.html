Post and Get in PHP

Usage:

Post is used when security matters for the requests made while Get is used whenever passed data are not private and need to be secured! We are speaking about the difference between passing credentials (needs post method) and any other type of data.

Arrays

We use the following function to iterate through arrays in PHP:

<?php
foreach($array_name as $item_name){
    echo"{$item_name}";
}
?>

We use the following functions to manipulate arrays in PHP:

<?php
array_push($array_name, "element");
array_pop($array_name); //to remove the last element
array_shift($array_name); //removes the first element and shifts the others
array_reverse($array_name); //returns a reversed array that needs to be assigned and printed to be seen
count($array_name); //displays the number of elements in the array
?>
Associative Arrays

They are more like dictionaries in Python, incorporating the key → value data storing type.

<?php
$array_name = array("key" => "value", ...);

// To iterate through
foreach($array_name as $key => $value){
    echo"{$key} = {$value} <br>";
}

// They have the same functions as normal arrays, except that there is a new one
array_flip($array_name); // makes values keys and keys values
?>
Input Filters

There are times where inputs might be inserted incorrectly or might contain an execution for a virus or an attack. To avoid this kind of problem, we can filter and sanitize input as follows:

<?php
if (isset($_POST['login'])) {
    $username = filter_input(INPUT_POST, "username", FILTER_SANITIZE_SPECIAL_CHARS);
    $age = filter_input(INPUT_POST, "age", FILTER_SANITIZE_NUMBER_INT);
    $email = filter_input(INPUT_POST, "email", FILTER_SANITIZE_EMAIL);
    
    echo"You are {$username} and {$age} years old! <br> Your email is: {$email}";
}
?>
Cookies

Cookies are used for storing information about the user. They contain key → value data plus two additional arguments!

<?php
setcookie("best_hobby", "Learning", time() + 86400, "/");
setcookie("best_friend", "AbuNaif", time() - 0, "/");
setcookie("best_info", "the hereafter", time() + (86400 * 3), "/");

foreach ($_COOKIE as $key => $value) {
    echo"{$key} = {$value} <br>";
}

if (isset($_COOKIE["best_friend"])) {
    echo"This was the name of your best friend: {$_COOKIE["best_friend"]} <br>";
} else {
    echo"I'm unable to detect who is your best friend";
}
?>

Hashing

Used to encrypt important data and hide it from 3rd parties by storing the password into symbols, letters, and characters after performing a mathematical operation (hashing algorithm).

<?php
$password = "Dawa2001";

$hash = password_hash($password, PASSWORD_DEFAULT);

if(password_verify($_POST["password"], $hash)){
    echo "You're logged in <br>";
    echo $hash;
} else {
    echo "Password is wrong";
}
?>
Connecting MySQL
<?php
// This is all written in the database PHP file to insert, not retrieve!
$db_server = "localhost";
$db_username = "root";
$db_password = "";
$db_database = "podcastdb";
$conn = "";

try {
    $conn = mysqli_connect($db_server, $db_username, $db_password, $db_database);
} catch(mysqli_sql_exception $exception) {
    echo "Couldn't establish a connection";
}

if ($conn) {
    echo "Connected successfully";
}
?>
